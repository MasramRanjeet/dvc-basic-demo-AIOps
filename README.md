# Commands used -

### Environment Creation 
```bash
conda  create -n env_name python=3.7 -y
```

### Activate Environment
```bash
conda activate env_name
```

or

```bash
source activate env_name
```

### Install dvc
```bash
pip install dvc
```

### initializing git
```bash
git init
```

### initializing dvc
```bash
dvc init
```

### reproduce dvc steps 
```bash
dvc repro
```

### create file 
```bash
touch filename.extenstion
```

### dag
```bash
dvc dag
```
